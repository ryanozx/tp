    Project: HRMate

    HRMate is a desktop address book application that aims to streamline HR processes, by offering an intuitive, CLI-based contact management system with specialised functionalities for HR tasks. It has a GUI created with JavaFX, and is written in Java with about 10 kLoC.

    • Documentation

    • New Feature: Import and export commands (#53, #58, #101)
      • Provides ability for users to import and export HRMate data in CSV format for greater portability
      • This feature provides greater interchangeability with other spreadsheet applications like Excel, and allows users to save filtered results that can be either stored elsewhere or sent to others
      • Highlights:
        • Created abstract classes for AdaptedPerson, AdaptedLeave, SerializableAddressBook and SerializableLeavesBook
        • Created CsvUtil to handle the reading and writing of CSV files
        • Created CsvFile to support the ability to query CSV files for values in a specified column by column name
      • New Feature: Find leaves by time and status (#90, #91)
      • Provides ability for users to filter leaves by either the time period or the leave status
      • This feature allows users to search for leaves more intelligently e.g. what leaves are occurring at a certain period, which leaves have not been approved yet
      • Highlights:
        • Created Range class to guarantee that the start dates will not occur after the end dates, and provided the option of whether to enforce non-null start and end dates
